# 0.1.1 — 2026-06-01

Initial public release. Provides MCP access to 1.8M+ IRS-recognized tax-exempt organizations and their Form 990 financial filings via the ProPublica Nonprofit Explorer API. No API key required.

## Added

- **`nonprofit_search`** — keyword search across 1.8M+ tax-exempt organizations with optional state, NTEE category, and subsection code filters; returns paginated results (25/page).
- **`nonprofit_get_organization`** — fetch full organizational profile by EIN, including latest Form 990 financials (revenue, expenses, assets, liabilities), officer list, and filing metadata.
- **`nonprofit_get_filings`** — paginated Form 990 filing history for an organization by EIN; each record carries tax year, form type, financials, and a ProPublica permalink.

## Changed

- **`formTypeLabel`** extracted from inline arrow to module-level function for clarity.
- **`McpError` instanceof guard** replaces string-code check in the service error re-throw path.
- **`svc.search(input, ctx)`** called directly instead of reconstructing an intermediate object.
- **`parseJson`** helper consolidates the HTML-masquerade check and JSON.parse error handling in the service.
- Metadata polished: `manifest.json` description and author populated; `server.json` runtime hints set to `bun`; `Dockerfile` description label and source label added; `.claude-plugin` and `.codex-plugin` plugin manifests populated.
